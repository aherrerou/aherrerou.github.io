Students Simulation with LLM Agents with LangChain and LangGraph

A self-contained Python script that uses LangChain and LangGraph to simulate a realistic high-school math class. Three agent types interact in a fixed number of “rounds”: - ModeratorAgent picks who speaks next - StudentAgent generates a short student reply - TeacherAgent responds and applies negative punishment if needed
